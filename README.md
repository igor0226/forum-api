# Modern python backend

## Technologies:
1) aiohttp
2) asyncpg
3) PostgreSql
4) jinja2

## Launch

1) Build docker images
```shell
./scripts/psql/build_docker.sh
```
```shell
./scripts/backend/build_docker.sh
```
```shell
./scripts/backend/build_docker.sh
```
2) Launch it
```shell
./scripts/psql/run_docker.sh
```
```shell
./scripts/backend/run_docker.sh
```
```shell
./scripts/backend/run_docker.sh
```

### App hosts
- frontend - 8080 port
- backend - 5000 port
