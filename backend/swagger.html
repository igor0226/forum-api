GET /forum/{slug}/users Пользователи данного форума

Description

Получение списка пользователей, у которых есть пост или ветка обсуждения в данном форуме.

Пользователи выводятся отсортированные по nickname в порядке возрастания. Порядок сотрировки должен соответсвовать побайтовому сравнение в нижнем регистре.

Parameters

Name Located in Description Required Schema
slug path Идентификатор форума. True identity string
limit query Максимальное кол-во возвращаемых записей. int32 number
since query Идентификатор пользователя, с которого будут выводиться пользоватли (пользователь с данным идентификатором в результат не попадает). identity string
desc query Флаг сортировки по убыванию. boolean

Responses

Code Description Schema
200 Информация о пользователях форума.
[
about (text string, optional):

Описание пользователя.

email (email string):

Почтовый адрес пользователя (уникальное поле).

fullname (string):

Полное имя пользователя.

nickname (identity string, optional):

Имя пользователя (уникальное поле). Данное поле допускает только латиницу, цифры и знак подчеркивания. Сравнение имени регистронезависимо.

]
404 Форум отсутсвует в системе.
message (string, optional):

Текстовое описание ошибки. В процессе проверки API никаких проверок на содерижимое данного описание не делается.
